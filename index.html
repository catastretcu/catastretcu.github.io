Projects

Spike Guard Lid


With drink spiking becoming a growing issue, this project is an interesting IoT approach to protecting your drink while on a night out. The Sike Guard lid uses two accelerometers to measure unwanted movement of your drink. A Raspberry Pi processes the data from the accelerometers and sends a message over MQTT to indicate that the lid has been moved in a suspicious way. My main area of work for the project was developing a mobile app to pair with the lid. The app can be used to arm a lid when planning to leave it unattended. It then checks the MQTT server periodically for spike messages and alerts the user if they are received or if the lid loses connectivity. An RGB LED is also used to indicate the status of the drink, in case the user forgot to use the app.

Multi-Sensing Rover


The aim of the project is to design and build an autonomous rover that could be operated in a remote location without direct supervision. The project is split into five separate modules that aim to give full function to the rover assembly. The modules are: Energy, Drive, Vision, Control, Command. My work was focused on the Drive module. The main components of the Drive module are a pair of DC motors, an Arduino, a switch-mode power supply and an optical-flow sensor. The Arduino microcontroller is used to receive commands from the Command module and to control the SMPS in order to send power to the DC motors accordingly. The optical flow sensor is used to collect data about the position of the rover. This data is then sent over UART to other rover modules.

Seismometer


An affordable seismometer powered by an Arduino microcontroller. The controller is used to process inputs from two sensor modules: a 3-axis accelerometer module and a handmade oscillating mass sensor. The accelerometer is mainly used to measure P-waves, the component of seismic waves that travel parallel to the surface of the Earth, while the oscillating mass sensor is used to measure S-waves, which travel perpendicular to the surface of the Earth. A neodymium magnet is connected to a spring and surrounded by a copper coil to make up the oscillating mass sensor. The magnet generates a current through the coil when it is oscillating and the Arduino, paired-up with a higher resolution ADC, is used to measure the current. Differential measurements from both the sensors are also used in an attempt to separate seismic events from environmental noise.
